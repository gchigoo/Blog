# 欢迎使用极简博客系统

这是一篇测试文章，展示博客的基本功能。

## 功能特性

- **Markdown 支持**: 使用标准 Markdown 语法编写文章
- **自动图片转换**: 图片自动转换为 WebP 格式，节省带宽
- **标签系统**: 支持多标签分类
- **归档功能**: 按时间自动归档
- **极简设计**: 基于 new.css，快速加载
